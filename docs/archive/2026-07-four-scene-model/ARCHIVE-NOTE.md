# 四场景产品模型归档说明

归档日期：2026-07-16。

本目录保存本轮重检前位于 `docs/` 根目录的完整当前文档。该模型以“在本机运行 / 管理版本 / 部署测试版 / 发布正式版”四个顶层场景组织产品，已经被新的领域模型替代。

主要失效原因：

- 把临时部署任务当成稳定导航，导致项目入口被失败和恢复状态劫持；
- “管理版本”同时承载一次性设置和版本目录，目的不清晰；
- 测试版、正式版与版本管理来回跳转，版本和环境仍然耦合；
- 配置中心大多只是一次性填值，没有形成项目 + 环境的持续绑定；
- 测试和正式服务器、全局连接与项目引用的边界不够明确。

当前有效模型请从 [`docs/README.md`](../../README.md) 开始。此目录仅用于追溯，不作为实现与验收依据。
